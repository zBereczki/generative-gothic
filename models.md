On this page some generated models can be found, embedded. Under each heading the models are generated by the same algorithm.

# Usage

**If the models are loaded too small or not centered, press Z. Sometimes the predefined views don't load correctly, this is a bug in the [Iris plugin](https://mcneel.github.io/Iris/).**

The models can be rotated by clicking inside the model window with the left mouse button (or trackpad), then dragging around while holding the mouse button (or trackpad). The models can be zoomed by scrolling, and panned by clicking and holding the right mouse button.
